Detailed Description

This class will handle the translations of the application.

This class is based on two qrc-files: languages.qrc and qt_languagges.qrc. Both files includes the qm-translation files for the application messages and the internal Qt messages. The qm-translations of Qt are located in: {QT-DIR}.X.X.Xcompiler} Qt or better QApplication references only on one intance of QTranslator. If you loose the reference to it, you can't handle language changes anymore. Furthermore you can't access this function from other classes until they get the pointer.
