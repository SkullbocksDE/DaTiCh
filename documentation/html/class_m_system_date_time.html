Detailed Description

This class is able to change the system datetime statically or to call a gui wizard.

This class can be used to change the system datetime. Here a short overview of the typical system commands of windows and unix for datetime changes which will be used by this class.

== WIN ==
Get Date: date /T
Get Time: time /T
Set Date: date 5/15/2009 || date 2/14/1999
Set Time: time 5:34:00 PM || time 12:15:00 AM || time 23:23:23
=========
== UNIX ==
Get Date: date
Get Time: hwclock --show --utc
Set Date: date +%Y%m%d -s "20081128"
Set Time: date +%T -s "10:13:13"
==========
